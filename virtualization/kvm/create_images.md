# Linux镜像
## RHEL/CentOS系统
- 镜像大小 20G左右即可，生产环境灵活控制，不要太大
- 分区
  - boot 1G
  - swap分，根据情况自行挂载
  - 其他都给/分区，根据业务要求挂载数据盘
- 系统升级
- 删除旧内核(节约空间)
- 安装基础组件和开发组件
- yum源配置
- NTP配置
- SElinux及iptables设置
- 其他自定义设置

> 所有系统要经过严格的性能及稳定性测试
