## PHP_CodeSniffer部署安装
``` bash
wget https://squizlabs.github.io/PHP_CodeSniffer/phpcs.phar
wget https://squizlabs.github.io/PHP_CodeSniffer/phpcbf.phar
install phpcs.phar /usr/bin/phpcs
install phpcbf.phar /usr/bin/phpcbf

phpcs --version
phpcbf --version
```
