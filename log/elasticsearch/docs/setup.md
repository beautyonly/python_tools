# java(JVM)版本
至少Java 8才能运行。只支持Oracle的java和openJDK。所有Elasticsearch节点和客户端都应该使用相同的jvm版本。

建议安装java版本 1.8.0_131+,如果使用已知java版本，Elasticsearch将拒绝启动。

Elasticsearch将使用的java版本可以通过设置JAVA_HOME环境变量进行设置
