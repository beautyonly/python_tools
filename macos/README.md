# Mac OS常用指令
## Mac快捷键大全
### Command
Command是Mac里最重要的修饰键，在大多数情况下相当于Windows下的Ctrl。
- Command-Z 撤销　
- Command-X 剪切　　
- Command-C 拷贝（Copy）　　
- Command-V 粘贴　　
- Command-A 全选（All）　　
- Command-S 保存（Save)　　
- Command-F 查找（Find）　

### Control
- Control-A 移至行或段落的开头
